## Slightly More Formal Version

Hi, I am sharing an early prototype of `Anestha`, a web app intended to bring commonly used anesthesia calculation tools into one place.

The current prototype includes:
- infusion calculations
- dilution calculations
- pediatric dosing / airway reference
- MH / dantrolene quick reference

This version is still under active review. It is meant for `reference and feedback` purposes and should not be used as a sole source for clinical decisions.

If you try it, I would really appreciate comments on:
- whether the UI feels intuitive
- whether any wording is unclear
- whether the mobile experience is usable
- whether any reference, range, or note looks wrong or misleading

The easiest way to send feedback is through the `Support` tab inside the app.
