Clinical Disclaimer

Anestha is an anesthesia calculator prototype intended for education, workflow review, and feedback gathering. Independent verification remains necessary.

What this app is

Anestha is a reference-oriented prototype that brings multiple anesthesia-related calculators into one interface for faster review and usability testing.

What this app is not

It is not a substitute for clinical judgment, institutional protocol, official prescribing information, manufacturer instructions, or emergency workflow policy.

  • do not use it as the sole basis for medication preparation
  • do not use it as the sole basis for dosing or infusion setup
  • do not use it as the sole basis for airway sizing or emergency treatment planning

Reference and range limitations

Some values shown in the app reflect label information, some reflect clinical practice patterns, and some reflect study-specific contexts.

Label, Clinical, and Study-specific references should not be interpreted as a single universal dosing standard. Always review the original source and your institutional protocol.

User responsibility

Users remain responsible for checking medication concentration, units, indication, patient context, compatibility, device fit, and current institutional policy.

This is especially important for high-risk calculations, pediatric dosing, airway decisions, and MH / dantrolene preparation.

Current review state

The app includes reference audits and review metadata, but it remains a prototype under active iteration.
